# Email_Analysis
This repository contains an analysis report and related artifacts documenting suspicious emails and URLs identified as potential phishing attempts


Purpose: To document and analyze phishing emails and their associated URLs for educational and security awareness purposes.

# Content
1) phishing_analysis.md:  Detailed report summarizing the email and URL analysis, including findings and recommendations.
2) Screenshot_Email.png: Screenshot of the phishing email received via Thunderbird.
3) Screenshot_404.png: Screenshot of the 404 error page from the linked URL.
4) Screenshot_VirusTotal.png: VirusTotal analysis results for the URL.
