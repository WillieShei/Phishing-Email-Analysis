# Phishing Email Analysis
This repository contains an analysis report and related artifacts documenting suspicious emails and URLs identified as potential phishing attempts


Purpose: To document and analyze phishing emails and their associated URLs for educational and security awareness purposes.

# Content
1) Analysis: Detailed report summarizing the email and URL analysis, including findings and recommendations.
2) Email screenshot: Screenshot of the phishing email received via Thunderbird.
3) Website Error page: Screenshot of the 404 error page from the linked URL.
4) VirusTotal screenshot: VirusTotal analysis results for the URL.
